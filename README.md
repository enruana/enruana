# Hey, I'm Felipe Mantilla 👋

**Electronic Engineer · Full Stack Developer · Technical Writer**
**Boyacá, Colombia 🇨🇴**

I build things that live between hardware and software — from microcontrollers and embedded systems to full-stack web platforms and AI-powered tools. Currently working as **Solutions Architect** at [Gorilla Logic](https://gorillalogic.com/), where I lead AI-focused initiatives across engineering teams.

I'm passionate about reverse-engineering, music technology, 3D environments, and pushing the boundaries of what AI can do as a development partner. When I'm not coding, I'm probably saving my plants with an Arduino or tearing apart some legacy protocol.
